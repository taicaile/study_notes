## 第二章 爬虫基础
### 请求方式
- get
	- get请求中的参数包含在URL里面, 数据可以在URL里看见。而post请求的URL不会包含这些数据，数据都是通过表单形式传输的, 会包含在请求体中。
- post
	- get请求提交的数据最多只有1024字节, 而post方式没有限制。
	
#### 请求头
- Accept
- Accept-Language
- Accept-Encoding
- Host
- Cookie
- Referer
- User-Agent
- Content-Type

### 响应
- 响应状态码
	- 200 表示服务器正常响应
	- 404 代表页面未找到
	- 500 代表服务器内部发生错误
- 响应头
	- Date
	- Last-Modified
	- Content-Encoding
	- Server
	- Content-Type
	- Set-Cookie
	- Expires
- 响应体
